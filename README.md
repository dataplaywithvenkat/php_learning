### String Concatenation

```php
echo "one" . "two"; // Prints: onetwo

echo "one " . "two"; // Prints: one two

echo "one" . " " . "two" . " " . "three"; // Prints: one two three

echo "\nMy name is:"." Code"."cademy";
  
echo "\n". "tur". "duck"."en";

// My name is: Codecademy
// turducken

```
